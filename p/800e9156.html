linux锁屏命令

列出当前所有的会话。这是默认命令。
sudo loginctl list-sessions

锁定/解锁会话(如果会话支持屏幕锁)。
如果指定了会话ID，那么将锁定/解锁指定的会话，否则将锁定/解锁当前调用者的会话。
sudo loginctl lock-session [ID…]
sudo loginctl unlock-session [ID…]

锁定/解锁所有支持屏幕锁的会话。
sudo loginctl lock-sessions
sudo loginctl unlock-sessions
